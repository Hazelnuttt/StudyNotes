## 第二章 let 和 const 命令

- 不同点：let 定义了一个变量，const 定义了一个常量
  其他：
- 都不能重复定义

### 补充

- 声明变量只在 let 命令所在“代码块”内有效/for 循环

```js
{
  let a = 10
  var b = 1
}
// console.log(a)
console.log(b)
```
